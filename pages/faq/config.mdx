# Configuration

Run `/config` to see the configuration command.

<Callout type="warning">Users must have administrator permissions to access this command.</Callout>
<Callout type="info">To hide the response, select `ephemeral` and set it to true.</Callout>

## Config Structure

![Config Structure](public/cfg-prompt.png)

Here's a list of available configurations:

1. [Config Status](#config-status)
2. [Activate Premium](#activate-premium)
3. [Set Channel](#channel-set)
4. [Remove Channel](#channel-remove)

## Config Status

To view the configuration status, select `Config Status` from the [Config Structure](#config-structure) prompt.

You will then be able to see the status as shown below:

![Status](public/cfg-status.png)

## Activate Premium

To activate the premium feature, select `Activate Premium` from the [Config Structure](#config-structure) prompt.

A modal will appear prompting you to enter the given code. After entering the code, you will see:

![Premium Modal](public/cfg-premium-prompt.png)

Once the code is successfully entered, you will see:

![Premium Activated](public/cfg-premium-done.png)

## Channel Set

To set the log channel, select `Channel Set` from the [Config Structure](#config-structure) prompt.

A list of channels will appear like this:
![Channel Set](public/cfg-channel-set.png)

Select a channel from this prompt, then you will see:
![Channel Done](public/cfg-channel-done.png)

## Channel Remove

To remove the log channel, select `Channel Remove` from the [Config Structure](#config-structure) prompt.

You will then see:
![Channel Removed](public/cfg-channel-remove.png)
